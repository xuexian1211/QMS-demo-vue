# 检验数据模型用户操作手册（更新）

## 功能入口
- 菜单路径：`检验主数据 -> 检验数据模型`

## 页面结构
- 工具栏：`新增`、`编辑`、`查看`、`删除`、`导出`、`刷新`
- 搜索条件：模型类型、编码、名称；不良现象支持严重等级筛选；不良原因支持原因大类筛选
- 数据列表：支持排序、分页、行选择

## 模型类型说明
- 缺陷分类（QM_DefectCategory）：分类代码、分类名称、父级ID、描述
- 不良现象（QM_DefectPhenomenon）：不良类型ID、现象代码、现象名称、严重等级、适用工序类型、描述
- 不良原因（QM_DefectCause）：原因代码、原因名称、父级ID、原因大类（5M1E）、是否高频

## 基本操作
- 新增：点击`新增`，在弹出的表单中按模型类型填写必填项后点击`确定`
- 编辑：勾选 1 条记录，点击`编辑`，修改后点击`确定`
- 查看：勾选 1 条记录，点击`查看`，查看详情提示
- 删除：勾选多条记录，点击`删除`并确认
- 导出：点击`导出`，导出当前筛选条件下的数据（后端接口接入后生效）
- 刷新：点击`刷新`，刷新当前数据

## 表单验证
- 缺陷分类：`分类代码`、`分类名称`为必填
- 不良现象：`不良类型ID`、`现象代码`、`现象名称`、`严重等级`为必填
- 不良原因：`原因代码`、`原因名称`、`原因大类`为必填；`是否高频`为开关

## 注意事项
- 不良现象的`不良类型ID`为缺陷分类中的`分类名称`所对应的`ID`
- 暂为前端模拟数据；接入后端接口后请遵循接口返回结构

## 响应式与风格
- 页面整体风格与“来料不合格处置单”一致，表单采用卡片分组、1400px宽度弹窗、紧凑表格与分页
