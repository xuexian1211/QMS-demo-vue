# 量检具详情页

## ADDED Requirements

### Requirement: 量检具详情页布局
系统 MUST 提供量检具详情页，采用上方基本信息表单+下方 Tab 行的布局结构（参考检验方案表单 InspPlanEdit.vue 风格）。

#### Scenario: 访问量检具详情页
- 当用户在量检具列表中点击"查看"操作时
- 系统 MUST 导航到量检具详情页
- 详情页 MUST 显示两个区块：上方基本信息 Card 和下方 Tab Card

#### Scenario: 查看基本信息区块
- 当用户进入详情页时
- 系统 MUST 在上方 Card 中以 Descriptions 组件展示量检具的所有字段信息
- 信息 MUST 包含：量具编号、名称、规格型号、制造商、量程、精度、校准周期、校准日期、状态、保管人、位置等

#### Scenario: 查看 Tab 区块
- 当用户进入详情页时
- 系统 MUST 在下方 Card 中显示 Tab 页：校准记录、状态变更、附件
- 用户 MUST 能够切换不同的 Tab 查看相应内容

#### Scenario: 详情页编辑功能
- 当质量工程师在详情页点击"编辑"按钮时
- 系统 MUST 打开编辑弹窗，允许修改量检具信息
- 保存后系统 MUST 刷新详情页内容

#### Scenario: 详情页快速校准
- 当质量工程师在详情页点击"校准"按钮时
- 系统 MUST 打开新增校准记录弹窗
- 保存后系统 MUST 刷新校准记录列表和基本信息中的校准日期
