# 检验计划管理开发提案 (Inspection Plan Management Proposal)

## 概述 (Overview)
本提案旨在实现质量主数据模块中的“质量检验计划与方案 (Inspection Plan/Scheme)”功能。检验计划是统筹 IQC、IPQC、FQC 与 OQC 等全工艺制程检验的纲领性顶层设计，它将检验项目、检验方法和 AQL 抽样规则按照逻辑组合并打包为一个供一线使用的一体化检验套件。

## 目标 (Goals)
- 实现检验计划的高级查询与状态总览（草稿、审批中、已批准、已作废）。
- 实现跨组织的多级过滤与权限控制（集团级与工厂级）。
- 实现独创的副本复制向导（Clone/Copy Modal），支持跨组织复制方案。
- 实现检验方案的工作流级生命周期管理。
- 实现检验模板的关联挂载与检验明细的深度定制。
- 设计完善检验计划的数据模型（InspectionPlan 表）。

## 非目标 (Non-Goals)
- 检验结果录入与判定逻辑（该部分属于质量执行工作流模块）。
- FMEA和MSA的深度业务计算交互（仅提供概念关联映射）。
