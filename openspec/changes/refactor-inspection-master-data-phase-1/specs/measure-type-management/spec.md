# 量检具类型管理

## ADDED Requirements

### Requirement: 独立的量检具类型管理页面
The system MUST provide an independent gauge type management page. 系统应提供独立的量检具类型管理页面

#### Scenario: 质量工程师访问量检具类型管理页面
- 当质量工程师访问"量检具类型管理"页面时
- 系统应显示量检具类型树
- 树应支持多级类型展示
- 树应显示类型名称、精度、MSA要求

#### Scenario: 质量工程师创建新的根级量检具类型
- 当质量工程师点击"新增根类型"按钮时
- 系统应显示新增类型表单
- 系统应允许用户输入类型名称、精度、是否需MSA
- 系统应在保存后刷新类型树并显示新创建的类型

#### Scenario: 质量工程师编辑量检具类型
- 当质量工程师在树节点上点击"编辑"按钮时
- 系统应显示编辑表单，预填充当前类型信息
- 系统应允许用户修改类型名称、精度、是否需MSA
- 系统应在保存后更新类型树中的类型信息
