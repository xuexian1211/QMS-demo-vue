# 检验模板管理

## ADDED Requirements

### Requirement: 检验模板管理功能
The system MUST provide inspection template management functionality. 系统应提供检验模板管理功能

#### Scenario: 质量工程师访问检验模板列表页面
- 当质量工程师访问"检验模板"页面时
- 系统应显示检验模板列表
- 系统应提供搜索表单，支持按模板编码、名称、状态、版本筛选
- 系统应显示表格列：模板编码、模板名称、版本、状态、创建时间、操作

#### Scenario: 质量工程师创建新的检验模板
- 当质量工程师点击"新增"按钮时
- 系统应跳转到检验模板编辑页面
- 系统应显示模板基本信息表单
- 系统应允许用户输入模板编码、模板名称、版本
- 系统应设置模板初始状态为"草稿"

#### Scenario: 质量工程师编辑检验模板明细
- 当质量工程师在模板编辑页面时
- 系统应显示模板明细表格
- 系统应允许用户添加、编辑、删除明细行
- 系统应允许用户配置检验项目、抽样规则、特性分类等
- 系统应支持明细行拖拽排序
