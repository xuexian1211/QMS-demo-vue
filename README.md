# S_QMS 1.0 - 汽车零配件质量管理系统

## 📋 项目简介

S_QMS 1.0 是专为汽车零配件企业设计的质量管理系统，基于Vue3 + TypeScript + Ant Design Vue构建的前端应用。

## 🚀 技术栈

- **Vue 3.4+** - 现代化前端框架
- **TypeScript 5.2+** - 类型安全
- **Ant Design Vue 4.2+** - 企业级UI组件库
- **Vite 5.0+** - 快速构建工具
- **Pinia 2.1+** - 状态管理
- **Vue Router 4.2+** - 路由管理
- **ECharts 5.4+** - 数据可视化
